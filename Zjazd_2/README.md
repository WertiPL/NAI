## Fuzzy logic example

Train braking simulation using fuzzy logic to calculate braking power by taking
distance to turn, turn radius and speed.

### Installation

From repository-level, execute:

```
python3 -m venv venv
source /venv/bin/activate
pip3 install -r requirements.txt
```

To run, go to task folder and start:

```
cd Zjazd_2
python3 main.py
```
